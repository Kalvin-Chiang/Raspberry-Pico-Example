AT24C256
===

### Pinout

我手邊目前只有這二種包裝

#### DIP

```
     +---------------+
 A0 ─┤ 1           8 ├─ Vcc
     |               |
 A1 ─┤ 2    DIP    7 ├─ WP
     |               |
 A2 ─┤ 3           6 ├─ SCL
     |               |
GND ─┤ 4           5 ├─ SDA
     +---------------+
```

#### TSSOP

24LC02B

```
      +--------------------------+
  A0 ─┤ o                        ├─ Vcc
  A1 ─┤          TSSOP           ├─ WP
  A2 ─┤                          ├─ SCL
 Vss ─┤          (TOP)           ├─ SDA
      +--------------------------+
```
